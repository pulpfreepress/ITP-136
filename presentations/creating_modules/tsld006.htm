Creating .Net Modules

  • From the command-line, first create the Dog module:

    • csc /target:module Dog.cs

  • The name of the file that’s created is

    • Dog.netmodule

  • Next, create the MainApp executable file using the /addmodule option:

    • csc addmodule:Dog.netmodule MainApp.cs
