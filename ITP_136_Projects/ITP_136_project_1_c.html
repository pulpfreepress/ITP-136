ITP 136
C# Programming

Project 1

 

Project Name: Console Calculator

Objectives:

  • Design and implement a C# console application
  • Understand and internalize the development cycle: design, code, test, refactor
  • Demonstrate your ability to compile and run C# programs from the command line
  • Demonstrate your understanding of keyboard input/output (IO), variables, constants, and fundamental control structures
  • Demonstrate your ability to debug code

Tasks:

  • Write a console application that adds, subtracts, multiplies, and divides integers and floating point numbers
  • The application should display a running total as you complete each operation
