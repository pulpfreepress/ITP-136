ITP 136 — Introduction to C# Programming
Project 3

Media Library Application

Objectives:

  • Utilize inheritance in your application design
  • Utilize polymorphic behavior in your application design
  • Create and use an abstract base class
  • Demonstrate your ability to inherit from an abstract base class and override derived class methods and properties
  • Create an application architecture based on the Model-View-Controller software design pattern

Tasks:

  • Create an application that lets you enter information about various types of media you might have in your house. Examples of these might include books, CDs, DVDs, etc.
  • As part of your design analysis determine the common properties shared by each media type and locate those in the base class. Extend the base class to create specialized subclasses representing each different type of media.
  • Declare an abstract readonly property named Description in the base class and override it in each of the subclasses.
  • Use a collection class to store and manipulate media objects.
  • Save and retrieve the collection class to/from disk using serialization.
  • Structure the applicatioin architecture upon the Model-View-Controller software design pattern
