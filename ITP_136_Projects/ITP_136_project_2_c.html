ITP 136 C# Programming
Project 2

Photo Database Application

 

 

Objectives:

  • Create a Windows Forms application
  • Create and manipulate bitmap images from gif, tiff, or jpeg files
  • Create user defined data types (classes and/or structures)
  • Persist data to disk using serialization
  • Utilize properties in class design
  • Manipulate a collection of objects

 

 

 

 

Tasks:

  • Write a program that lets you upload images in the form of gif, tiff, and jpeg files and add descriptive information about each image.
  • Design a class that represents the image and its associated properties.
  • Use a collection class to store objects of this type.
  • Save and retrieve the collection of image objects to/from disk using object serialization. (This may require converting the image into an array of bytes.)
  • Create a user interface for the application that allows users to select images to upload, add descriptive data, and save the collection of images to disk.
